# HydroSpy Instrucciones

# COPY CODE Copiar El Código A Tu Portapapeles
# EXECUTE Ejecutará El RemoteEvent Seleccionado
# LOOP ON/OFF Repetirá El Código Cada Segundo Sin lag
# BLOCK RED Impedirá Que El RemoteEvent Se Ejecute
# CLEAR LOGS limpiar El panel De La Izquierda
# BUCLES Mostrará Todos Los LOOPS Activos
